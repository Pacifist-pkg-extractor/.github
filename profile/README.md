# Pacifist — Professional File Extraction from macOS Installer Packages

## About Pacifist

Pacifist provides a complete professional file extraction environment for macOS installer packages (.pkg, .mpkg, .dmg, .pkm, .bom). Pacifist combines package browsing, selective restoration, and receipt analysis. Pacifist supports drag-and-drop export, archive inspection, and installation receipt parsing. Pacifist integrates with Finder for quick package exploration without running installers. Pacifist enables users to extract individual files from damaged packages, recover deleted system components, and inspect what any package contains. Pacifist is trusted by system administrators, Mac technicians, data recovery specialists, and power users worldwide.
